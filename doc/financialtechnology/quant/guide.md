# golang
可以使用第三方软件包，如：

1. gostock：它是Go语言的A股数据抓取和分析库，可以获取上证指数、深证指数、创业板指数等A股数据；

2. go-finance：它是一个Go语言的金融库，可以获取A股数据，支持上证指数、深证指数等；

3. go-sina-finance：它是Go语言的新浪财经模块，可以获取A股数据，包括股票列表、报价、K线等。
